Sobre o Projeto

O site Marketing Criativo é um projeto de uma agencia de Marketing fictícia criada apenas para portfólio, todo o design contido neste projeto foi criado do zero com base em conceitos de Interface de Usuários e Experiência de usuários, podendo se utilizar do site em qualquer dispositivo.

Prototipação

O protótipo foi realizado na ferramenta Figma, foi levado em consideração toda ambientação do template para web e mobile, adaptando o estilo de alguns elementos de acordo com a configuração da tela. Clique nos links no final desta páginapara visualizar o protótipo web e mobile, a apresentação do projeto ou até mesmo o código fonte.

Desenvolvimento

O site foi desenvolvido sem a utilização de frameworks, o intuito é a fixação de conhecimentos conceituais do desenvolvimento web, com isso foi utilizado apenas html, css e javaScript, apenas no deploy do site foi utilizado a ferramenta online netlify.
